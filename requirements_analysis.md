# 파이썬 코드 설명 생성기 - 요구사항 분석

## 프로젝트 개요
파이썬 코드를 첨부하면 초등학생 수준과 컴퓨터 공학 1학년 수준으로 단계별 해설을 제공하는 인터랙티브 웹 애플리케이션

## 핵심 기능

### 1. 코드 입력 방식
- 파이썬 파일(.py) 업로드
- 텍스트로 직접 코드 입력

### 2. 설명 수준 (2단계)
#### 레벨 1: 초등학교 고학년 (12-13세) 수준
- 비유를 적극 활용한 쉬운 설명
- 모든 코드 요소를 하나하나 언급
- "(파이썬아)" 형식의 대화체 설명
- 내장/외장 모듈, 함수, 자료형 등 모든 요소 세세히 설명

#### 레벨 2: 컴퓨터 공학과 1학년 수준
- 정밀하고 구체적인 기술적 설명
- 전문 용어 사용
- 코드의 동작 원리와 구조 설명

### 3. 설명 대상 요소
- 내장 모듈 및 외장 패키지
- 함수 (내장 함수, 사용자 정의 함수)
- 자료형 (문자열, 숫자, 리스트, 딕셔너리 등)
- 제어문 (조건문, 반복문)
- 변수와 할당
- 연산자
- 클래스와 객체 (있는 경우)

### 4. UI/UX 요구사항
- 시각적으로 인터랙티브한 웹 인터페이스
- 코드 입력 영역
- 두 가지 수준의 설명을 구분하여 표시
- 깔끔하고 읽기 쉬운 레이아웃

## 기술 스택 (예정)
- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- AI/LLM: OpenAI API (코드 분석 및 설명 생성)
- 배포: GitHub Pages 또는 로컬 실행

## 예시 설명 형식

### 예제 코드:
```python
import pandas as pd
pd.read_excel("Users/Downloads/around.xlsx")
```

### 레벨 1 설명:
"(파이썬아) 판다스라는 외부 패키지를 불러와줘. 그리고 그걸 앞으로 판다스를 pd라는 이름으로 줄여서 쓸게.
(파이썬아) pd(판다스) 안에 있는 'read_excel'이라는 '함수'를 사용해서, 다운로드 폴더 안에 있는 'around.xlsx'라는 이름의 엑셀 파일의 내용을 읽어줘."

### 레벨 2 설명:
"pandas 라이브러리를 import하여 pd라는 별칭으로 사용합니다. pandas는 데이터 분석을 위한 강력한 라이브러리입니다. read_excel() 메서드를 호출하여 지정된 경로의 Excel 파일을 DataFrame 객체로 읽어들입니다."
