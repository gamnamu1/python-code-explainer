# 파이썬 코드 설명 생성기 사용 가이드

## 프로젝트 개요

**파이썬 코드 설명 생성기**는 파이썬 코드를 입력하면 AI가 자동으로 분석하여 두 가지 수준의 설명을 제공하는 웹 애플리케이션입니다. 코딩 지식이 부족한 초보자부터 기술적 이해가 필요한 전공자까지, 각자의 수준에 맞는 설명을 받을 수 있습니다.

## Powered by Manus

이 웹사이트는 최첨단 기술 스택으로 구축되었습니다.

**프론트엔드**: React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui 컴포넌트 라이브러리를 활용한 현대적이고 반응형 사용자 인터페이스

**백엔드**: Express 4 + tRPC 11을 통한 타입 안전 API, Manus OAuth 기반 인증 시스템

**데이터베이스**: MySQL/TiDB + Drizzle ORM을 통한 안정적인 데이터 관리

**AI 엔진**: Manus 내장 LLM API를 활용한 고급 코드 분석 및 자연어 설명 생성

**배포**: 자동 스케일링 인프라와 글로벌 CDN을 통한 빠르고 안정적인 서비스 제공

## 웹사이트 사용 방법

### 1. 파이썬 코드 입력

파이썬 코드를 입력하는 방법은 두 가지입니다. 첫 번째는 "파이썬 파일(.py)을 클릭하여 업로드" 영역을 클릭하여 컴퓨터에 저장된 `.py` 파일을 선택하는 것입니다. 파일을 선택하면 자동으로 코드가 텍스트 영역에 표시됩니다. 두 번째는 텍스트 영역에 직접 파이썬 코드를 입력하거나 붙여넣는 방법입니다. 어떤 방법을 사용하든 코드가 입력되면 글자 수가 우측 하단에 표시됩니다.

### 2. 코드 분석 실행

코드를 입력한 후 "코드 분석하기" 버튼을 클릭합니다. AI가 코드를 분석하는 동안 버튼은 "분석 중..."으로 변경되며, 우측 설명 결과 영역에 로딩 애니메이션이 표시됩니다. 분석이 완료되면 자동으로 두 가지 수준의 설명이 생성됩니다.

### 3. 설명 확인 및 전환

설명 결과 영역 상단에는 "초등학생 수준"과 "전공자 수준" 두 개의 탭이 있습니다. "초등학생 수준" 탭을 클릭하면 비유와 친근한 말투로 작성된 쉬운 설명을 볼 수 있습니다. 이 설명은 코딩 개념을 처음 접하는 사람도 이해할 수 있도록 마치 이야기를 들려주듯 작성되었습니다. "전공자 수준" 탭을 클릭하면 기술적 용어와 개념을 사용한 전문적인 설명을 볼 수 있습니다. 이 설명은 컴퓨터 공학을 공부하는 학생이나 개발 경험이 있는 사람에게 적합합니다.

### 4. 코드 하이라이팅 및 복사

설명 내에 포함된 코드 블록은 자동으로 색상이 입혀져(하이라이팅) 읽기 쉽게 표시됩니다. 각 코드 블록 우측 상단에는 복사 버튼이 있어, 클릭하면 해당 코드를 클립보드에 복사할 수 있습니다.

## 웹사이트 관리

### 데이터베이스 관리

우측 Management UI에서 "Database" 패널을 열면 저장된 코드 분석 이력을 확인하고 관리할 수 있습니다. `code_analyses` 테이블에는 사용자가 분석한 모든 코드와 생성된 설명이 저장됩니다. 필요한 경우 특정 분석 결과를 삭제하거나 수정할 수 있습니다.

### 설정 변경

"Settings" 패널의 "General" 섹션에서 웹사이트 이름(`VITE_APP_TITLE`)과 로고(`VITE_APP_LOGO`)를 변경할 수 있습니다. 변경 사항은 즉시 웹사이트에 반영됩니다.

### 코드 확인

"Code" 패널에서 프로젝트의 전체 파일 구조를 확인하고, 필요한 경우 모든 파일을 다운로드할 수 있습니다. 주요 파일은 다음과 같습니다:
- `server/routers.ts`: 코드 분석 API 로직
- `client/src/pages/Home.tsx`: 메인 페이지 UI
- `drizzle/schema.ts`: 데이터베이스 스키마
